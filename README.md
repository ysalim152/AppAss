# AppAss - Gestion Association Sportive

## 📌 Description
Application web permettant de gérer une association sportive :
- Gestion des membres
- Gestion des équipes
- Gestion des séances
- Planning des activités

## 🛠️ Technologies
- HTML5 / CSS3 / JavaScript
- PHP 8
- MariaDB

## ⚙️ Installation
1. Cloner le projet
2. Importer database/association.sql
3. Configurer config/db.php
4. Lancer avec XAMPP ou Apache

## 🔐 Accès Admin
Créer un utilisateur dans la table users avec password hashé
